      이 사이트의 장점

      AI File Tools는 ChatGPT, Claude, Gemini 같은 AI 도구에 자료를 넣기 전에 파일을 정리할 수 있는 무료 문서 변환 사이트입니다. PDF 용량을 줄이고, 필요한 페이지만 추출하고, 여러 문서를 합치고, 이미지나 텍스트를 PDF로 바꿀 수 있습니다.

      가장 큰 장점은 파일을 서버로 업로드하지 않는다는 점입니다. 파일 선택은 브라우저 안에서 이루어지고, JavaScript가 사용자의 기기에서 직접 처리합니다.

      AI 작업 전에 필요한 이유

      AI에 파일을 업로드할 때 용량 제한, 페이지 제한, 파일 형식 제한을 만나는 경우가 많습니다. 이럴 때 PDF 압축, 분할, 병합, 이미지 PDF 변환 기능을 사용하면 자료를 더 깔끔하게 준비할 수 있습니다.

      이미지 속 글자를 OCR로 뽑아 텍스트로 만들면 AI에게 내용을 붙여 넣어 요약, 번역, 정리, 표 변환 등을 요청하기 쉽습니다.

      PDF 압축이 필요한 경우

      스캔본, 이미지가 많은 PDF, 과제 제출용 문서, 이메일 첨부 문서의 용량이 클 때 PDF 압축 기능이 유용합니다. 낮은 압축은 품질을 우선하고, 강한 압축은 용량 감소를 우선합니다.

      텍스트 위주의 PDF는 이미 용량이 작을 수 있어 압축 효과가 작을 수 있습니다. 또한 압축 결과는 이미지 PDF가 될 수 있어 텍스트 선택이 어려울 수 있습니다.

      이미지와 텍스트 변환

      이미지 PDF 변환 기능은 여러 장의 사진, 캡처 이미지, 스캔 이미지를 하나의 PDF로 만들 때 좋습니다. TXT PDF 변환 기능은 메모장 파일이나 정리한 원고를 보기 좋은 PDF로 저장할 때 사용할 수 있습니다.

      OCR 기능은 이미지 속 글자를 텍스트로 추출합니다. OCR은 브라우저에서 실행되기 때문에 큰 이미지는 처리 시간이 오래 걸릴 수 있습니다.

      자주 묻는 질문

      파일이 서버에 올라가나요?
      아니요. 파일은 브라우저 안에서 처리됩니다.

      OCR은 무엇인가요?
      이미지 속 글자를 인식해서 텍스트로 바꾸는 기능입니다.

      TXT를 OCR로 바꾸나요?
      TXT는 이미 텍스트이므로 OCR이 아니라 PDF 변환 기능으로 처리합니다.

      이 사이트는 PDF와 이미지 처리 기능을 브라우저에서 실행합니다. 사이트 운영자는 사용자가 선택한 파일의 내용을 볼 수 없습니다.

      단, pdf.js, pdf-lib, Tesseract.js 같은 외부 라이브러리는 CDN에서 불러옵니다. 향후 광고 서비스가 적용되면 쿠키가 사용될 수 있습니다.
